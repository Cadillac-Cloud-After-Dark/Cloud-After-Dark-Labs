## Overview

In this lab you will create a user. 

## What is a user?

👨🏾‍💻 A user can be defined as an entity. 

👨🏾‍💻 This entity can manipulate files and do other operations. 

👨🏾‍💻 Each user gets assigned an unique ID.

Let's create a user!

Command: 

sudo useradd "UserName"
